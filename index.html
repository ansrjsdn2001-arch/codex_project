Architecture

아키텍트 소개(나에 대한 소개로 변경)

스스로 지향하는 코딩 방향성에 대한 나의 소개 작성
현대적인 애플리케이션을 위한 견고한 토대를 만드는 것에 관심이 많습니다. 비즈니스 요구사항을 안정적인 API와 명확한 데이터 흐름으로 풀어내며, 유지보수하기 쉬운 서버 구조를 지향합니다.

  • 내가 구현, 설계 가능한 개념 설명
  • 객체 지향 설계와 계층형 아키텍처 이해
  • REST API 설계와 예외 처리 경험
  • MySQL, Redis 기반 데이터 관리
  • Docker와 AWS를 활용한 배포 경험
